# URL-Shortener
URL shortener web application that allows users

      To view a list of their created short URLs,
      Provide options to edit or delete existing short URL, and
      Display additional information, such as creation date and usage statistics.


# Create/Setup 

Virtual Environment
      python -m venv venv
      venv\Scripts\activate.ps1

# Installation
 
      python -m pip install Django

# Run Project

      python manage.py runserver
